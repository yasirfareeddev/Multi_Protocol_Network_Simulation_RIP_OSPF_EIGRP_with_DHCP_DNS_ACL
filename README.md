# Multi_Protocol_Network_Simulation_RIP_OSPF_EIGRP_with_DHCP_DNS_ACL
A comprehensive Cisco Packet Tracer simulation integrating RIP, OSPF and EIGRP with mutual route redistribution. Features centralized DHCP/DNS, cross-protocol Email services (SMTP/POP3), Wireless AP configuration and network security via Access Control Lists (ACL).
